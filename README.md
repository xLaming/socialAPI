# socialAPI
My simple social API done in PHP

# Functions:
  * SocialAPI::getInstagram($username)
  * SocialAPI::getTwitter($username)
  
# Calling:
  * You can use ?s=**SOCIAL_NAME**&u=**PROFILE_USER**
    * **SOCIAL_NAME** is the social networking
    * **PROFILE_USER** is the username
